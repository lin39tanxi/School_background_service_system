## 我重生了
我重生了。上一世，他一直把司婉当妹妹，我也没有把她当成眼中钉，现在细想来，好像我每次与他闹矛盾，司婉都会从中作梗。嘶，这个人真是不简单。于是，我简单打扮了一下，去了他爸妈住的老宅-----司婉为了取代我，是不是就会去老宅，以前我只当她是有孝心，毕竟他父母待她如亲女儿，现在我算是明白了，一切都是蓄谋已久，怪不得他父母总是在我面前讲起她的好，想到这里我心里很不舒服。她...真的是这样的人吗？思绪拉回，司机老王驾驶的车已缓缓驶入老宅，与此同时，司婉刺耳的声音传入我耳中：“时屿哥哥最近可辛苦了，最近我去探班过几次，他都忙的焦头烂额，可惜身边也没有一个知心人说体己话，洛熙姐姐也...”“爸妈，来看你们啦”她还没说完就被我打断。他爸妈见我来了，就忙示意司婉别再说了，并一边招呼我：“熙熙啊，今天怎么有时间过来了。”“时屿刚刚去了公司，这不一有时间就来看你们吗，刚好我顺路买了点水果，爸妈还有司婉你们都来吃。”司婉一脸不悦：“洛熙姐姐来看爸妈就买点水果啊，我来可是给叔叔阿姨买了燕窝补品呢。”“都是朋友，这么客气干嘛，不过我听说最近燕窝有造假的，吃起来对身体还有坏处呢。”我不露声色的回怼。“行了，今天也怪热闹的，大家聚起来吃个午饭吧。”他爸妈出来打圆场。“我当然可以啦，只不过洛熙姐姐这么忙，有时间吗？”“当然，陪爸妈吃顿饭当然还是有时间的。” 一顿饭后，我发现我似乎高估了司婉，她最多也就算一个白痴绿茶。至于阴谋陷害，凭她，估计是想不出来的。

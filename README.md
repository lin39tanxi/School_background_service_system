## 我重生了
我重生了。上一世，他一直把司婉当妹妹，我有把她当成眼中钉，现在细想来，好像我每次与他闹矛盾，司婉都会从中作梗。嘶，这个人真是不简单。于是，我简单打扮了一下，去了他爸妈住的老宅-----司婉为了取代我，是不是就会去老宅，以前我只当她是有孝心，毕竟他父母待她如亲女儿，现在我算是明白了，一切都是蓄谋已久，怪不得他父母总是在我面前讲起她的好，想到这里我心里很不舒服。她...真的是这样的人吗？思绪拉回，司机老王驾驶的车已缓缓驶入老宅，与此同时，司婉刺耳的声音传入我耳中：“时屿哥哥最近可辛苦了，最近我去探班过几次，他都忙的焦头烂额，可惜身边也没有一个知心人说体己话，洛熙姐姐也...”“爸妈，来看你们啦”她还没说完就被我打断。他爸妈见我来了，就忙示意司婉别再说了，并一边招呼我：“熙熙啊，今天怎么有时间过来了。”“时屿刚刚去了公司，这不一有时间就来看你们吗，刚好我顺路买了点水果，爸妈还有司婉你们都来吃。”司婉一脸不悦：“洛熙姐姐来看爸妈就买点水果啊，我来可是给叔叔阿姨买了燕窝补品呢。”“都是朋友，这么客气干嘛，不过我听说最近燕窝有造假的，吃起来对身体还有坏处呢。”我不露声色的回怼。“行了，今天也怪热闹的，大家聚起来吃个午饭吧。”他爸妈出来打圆场。“我当然可以啦，只不过洛熙姐姐这么忙，有时间吗？”“当然，陪爸妈吃顿饭当然还是有时间的。” 一顿饭后，我发现我似乎高估了司婉，她最多也就算一个白痴绿茶。至于阴谋陷害，凭她，估计是想不出来的。!
我又重生了，重生在他最爱我的那一年。上一世，我们先婚后爱，相濡以沫。在他生日的那一天，我在取蛋糕的路上出了车祸。那天，他等了我很久，知道我车祸的消息传入他耳中，他仍难以置信，随后抱头痛哭。我的灵魂在他身旁飘转。看见他哭，我的心也在隐隐作痛。“要是再来一次，我们能否相伴一生呢”猛然间，我眼前一黑。再次睁开眼，我的他正轻抚我的头，温柔的叫我起床。我激动的抱住他，哭着：“这是真的吗，我有见到你了，我们要永远在一起，这一回谁都不能把我们分开。”他愣了楞，但随即拍着我的背，轻声说着好。我环顾熟悉的一切，心中冒出一个想法：这一次我一定要和他好好的。但怎么会这么巧呢？怎么会我刚好出门买蛋糕，就刚好有司机撞了我呢。这一定是个阴谋。可我向来与世无争，怎么会...究竟是谁？我一把拉住他：“今天是几号”“睡糊涂啦，傻瓜，今天是六月七号。”他轻笑，将我拥入怀。闻着熟悉的味道，我的泪水不禁落下，并暗下决心：还有一个月，我一定不能重蹈覆辙。“亲爱的，最近公司那边怎么样了？与对手公司争的项目，争赢了吗”我和他有个约定，我们结婚之后，我们携手操办的这家公司要进入世界500强，还不错，目前算是小有名气了。“争赢了，公司最近很好，好的有点不正常呢，哈哈，对手公司那边气坏了，管他呢？项目一到手，再一办好，就差不多是500强了。”我回想起来，我死的那天，碰巧离500强申请通过还有三天，被撞时，好像隐约看到那个司机带了口罩？但开自己的车出去，为什么要戴口罩呢？我拍了拍脑袋，不行，太乱了。我翻身起床，催促他去公司：“最近应该挺忙的吧，快去公司吧，今晚早点回来。”他有点蒙，但还是乖乖听话，走前在我额上落下一个吻，出了门。他走后，我拿出纸和笔，将刚刚的思绪重新捋了一遍。等等，我好像还忘了一个点-----他的青梅竹马司婉。司婉一直看我不顺眼，曾想将我的位置取而代之，会不会是她呢？
